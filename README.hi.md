TelunKey एक देशी macOS शॉर्टकट लॉन्चर है जो एक कुंजी प्रवाह में ऐप सक्रियण और विंडो चयन को पूरा करता है।

- तेज प्रतिक्रिया, कम ओवरहेड और न्यूनतम रुकावट के लिए देशी स्विफ्ट के साथ निर्मित
- लेफ्ट/राइट कमांड (0 विलंब या कस्टम विलंब), लेफ्ट/राइट विकल्प (0 विलंब या कस्टम विलंब), और स्पेस ट्रिगर (सामान्य टाइपिंग को प्रभावित करने से बचने के लिए न्यूनतम 0.2 सेकेंड विलंब) का समर्थन करता है।
- उच्च-आवृत्ति वर्कफ़्लो और स्मूथ विंडो स्विचिंग के लिए अनुकूलित
- डिज़ाइन द्वारा स्थानीय-प्रथम, क्लाउड व्यवहार विश्लेषण पर डिफ़ॉल्ट निर्भरता के बिना

## सिस्टम आवश्यकताएं

- macOS 14.0 या बाद का संस्करण

## डाउनलोड करना

- नवीनतम रिलीज़ (डीएमजी): https://github.com/telungit/TelunKey/releases/latest/download/TelunKey.dmg
- सभी रिलीज़: https://github.com/telungit/TelunKey/releases

## स्थापना

1. डाउनलोड करें और `TelunKey.dmg` खोलें
2. `TelunKey.app` को एप्लिकेशन में खींचें
3. अभी ऐप न खोलें। पहले DMG के अंदर `一键修复.command` पर डबल-क्लिक करें
4. मरम्मत पूरी होने के बाद TelunKey खोलें
5. केवल स्क्रिप्ट विफल होने पर, टर्मिनल में मैन्युअल रूप से निम्न कमांड चलाएँ

```bash
xattr -dr com.apple.quarantine /Applications/TelunKey.app
```
