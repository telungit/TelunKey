TelunKey — это нативный лаунчер быстрых клавиш для macOS, который объединяет активацию приложений и выбор окон в одной последовательности клавиш.

- Написан на нативном Swift: быстрый отклик, низкая нагрузка и минимум вмешательства
- Поддерживает левый/правый Command (задержка 0 или пользовательская), левый/правый Option (задержка 0 или пользовательская) и триггер через Space (минимум 0,2 секунды, чтобы не мешать обычному набору текста)
- Оптимизирован для высокочастотных рабочих процессов и более плавного переключения окон.
- Local-first по задумке, без обязательной зависимости от облачной поведенческой аналитики.

- Все выпуски: https://github.com/telungit/TelunKey/releases.

## Установка

1. Загрузите и откройте TelunKey.dmg.
2. Перетащите TelunKey.app в приложения.
3. Пока не запускайте приложение. Сначала дважды щёлкните `2. Run Once After Install.command` внутри DMG.
4. После завершения исправления откройте TelunKey.
5. Только в случае сбоя сценария выполните следующую команду вручную в Терминале.

```bash
xattr -dr com.apple.quarantine /Applications/TelunKey.app
```

## Разрешения

Для полной функциональности TelunKey требуются следующие разрешения:

- Универсальный доступ: отслеживание глобальных событий клавиатуры.
- Запись экрана: создание миниатюр окон.

## Конфиденциальность

TelunKey придерживается подхода local-first. Основные данные остаются на вашем компьютере.
