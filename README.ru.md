TelunKey — это встроенный ярлык запуска macOS, который выполняет активацию приложения и выбор окна одним нажатием клавиши.

- Создан на основе Swift для быстрого реагирования, низких накладных расходов и минимальных прерываний.
- Поддерживает команду «Влево/Вправо» (задержка 0 или пользовательская задержка), опцию «Влево/Вправо» (задержка 0 или пользовательская задержка) и триггер пробела (задержка минимум 0,2 секунды, чтобы не влиять на обычный набор текста)
- Оптимизирован для высокочастотных рабочих процессов и более плавного переключения окон.
- Локальный подход по умолчанию, без использования по умолчанию облачной аналитики поведения.

## Установка

1. Загрузите и откройте TelunKey.dmg.
2. Перетащите TelunKey.app в приложения.
3. Пока не запускайте приложение. Сначала дважды щёлкните `一键修复.command` внутри DMG.
4. После завершения исправления откройте TelunKey.
5. Только в случае сбоя сценария выполните следующую команду вручную в Терминале.

```bash
xattr -dr com.apple.quarantine /Applications/TelunKey.app
```

## Разрешения

Для полной функциональности TelunKey требуются следующие разрешения:

- Доступность: слушайте глобальные события клавиатуры.
- Запись экрана: создание миниатюр окон.

## Конфиденциальность

TelunKey придерживается стратегии «сначала местные». Основные данные остаются на вашем компьютере.
